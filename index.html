RestAssured API Automation Framework

Java-based API testing framework using RestAssured, TestNG, Maven and Allure Reporting

View Code on GitHub

Project Overview

This project demonstrates an API automation testing framework designed to validate API endpoints, request payloads, response status codes, and test execution results. The framework follows a modular layered structure to keep the test code clean, reusable, and maintainable.

Tools & Technologies

Java RestAssured TestNG Maven JSON API Testing Allure Reports GitHub

Framework Architecture

APIRequestBuilder

Builds API requests by handling base URLs, headers, query parameters, and authentication setup.

Requests Class

Manages API calls such as GET, POST, PUT, and DELETE while capturing responses and status codes.

Payload Builder

Creates reusable request bodies for POST and PUT requests, supporting cleaner test data handling.

Test Structure

Uses organized packages for API tests, smoke tests, helper methods, and reusable utilities.

Allure Integration

Provides clearer test reporting with step-by-step traceability of requests, responses and results.

What This Project Shows

  • Ability to build and structure an API automation framework
  • Understanding of API request and response validation
  • Use of Java, RestAssured, TestNG, and Maven
  • Clean separation between test logic, request handling, and payload creation
  • Awareness of reporting and debugging through Allure
